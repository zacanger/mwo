## Usage:

```
npx mwo [foo]

~ mwo type
뭐? builtin

~ mwo ~
뭐? /home/z/: directory

~ mwo python3
뭐? /usr/bin/python3
```

You can install it globally, if you really want to, with `npm i -g mwo`.
